What is anchor tag ?

In HTML, an anchor tag is written as <a> and it is used to create hyperlinks. It connects one page to another, or links to external websites, files, emails, or even specific sections within the same page.

Example

<a href="https://example.com">Click Here <a>

Attributes of Anchor tags

It has several attributes that control how the link behaves.

1:Href

The href attribute in the anchor tag <a> stands for Hypertext Reference. It tells the browser where the link should go when the user clicks on it.

Example 
 <a href="https://www.example.com">Visit Example</a>
( Example)

This link will take you to "Example.com" when clicked.

2:Target

The target attribute in the <a> tag tells the browser where to open the link when a user clicks it

  • _self Default. Open in the same tab.
  • _blank:Open in a new tab.
  • _parent: Open in the parent frame.
  • _top: Open in the full body of the window.

Example

<a href="https://example.com" target="_blank">Open in New Tab</a>
(Open in New Tab)

3:Download

The download attribute in the <a> (anchor) tag is used when you want the user to download a file instead of opening it in the browser.

Example

<a href="example.pdf" download>Download pdf</a>
(Download pdf)

This link will download the file"example.pdf".

4:Rel

The rel attribute in the <a> tag defines the relationship between the current page and the linked page/document. It is very useful for SEO, security, and link behavior

  • nofollow: Tells search engines not to follow the link.</li>
  • noopener: Prevents the new tab from accessing the original page's context.
  • noreferrer: Prevents sending referrer information.

    Example

    <a href="https://www.example.com" rel="nofollow noopener"> Visit Example <a>
(Visit Example)

    5:Tittle
