What is html?

Ans: HTML (HyperText Markup Language) is the standard markup language used to create and design the structure of webpages.It organizes web content using elements and tags so that browsers can display text, images, links, and other multimedia properly.

History of html

1989:Tim Berners-Lee proposed the World Wide Web.

1991:First Version of HTML was introduced(HTML 1.0).

1995:HTML 2.0 was standardrized by the IETF.

1997:HTML 4.0 introduced more complex web features.

2014:Now HTML 5 became the latest standard with multimedia support, semantic elements, and mobile-friendly design.

What is Domain?

Ans:A domain is the unique name that identifies a website on the internet. It is the human-readable address used to access websites instead of typing their IP address.


History of domain

1. Before Domains (1960s–1980s):In the early days of the internet (ARPANET), computers were identified only by IP addresses (like 192.168.1.1).

2.Birth of the Domain Name System (DNS) – 1983:Paul Mockapetris invented the Domain Name System (DNS) in 1983. DNS made it possible to use easy-to-remember names instead of hard numeric IPs.
